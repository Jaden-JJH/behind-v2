# 후속 기사 타임라인 - 관리자 가이드

후속 기사 타임라인 기능을 사용하여 이슈의 시간 순서에 따른 관련 기사들을 관리하는 방법을 설명합니다.

---

## 📌 기본 사용법

### 1. 후속 기사 추가

1. **어드민 페이지** (`/admin`) 접속
2. 이슈 선택 → **"후속 기사 추가"** 버튼 클릭
3. 기사 정보 입력:
   - **제목** (필수, 1-200자)
   - **URL** (필수)
   - **기사 타입** (필수): 뉴스, YouTube, Twitter, Instagram
   - **설명** (선택): 하이라이트 기사에만 표시
   - **출처** (선택)
   - **썸네일 URL** (선택): 하이라이트 기사에만 표시
   - **하이라이트 표시** (선택): 중요 기사 강조

### 2. 후속 기사 수정

1. 타임라인에서 **"수정"** 버튼 클릭
2. 내용 변경 후 **"저장"**

### 3. 후속 기사 삭제

1. 타임라인에서 **"삭제"** 버튼 클릭
2. 확인 모달에서 **"삭제"** 클릭
⚠️ 삭제된 기사는 복구할 수 없습니다.

### 4. 순서 변경

1. 드래그 핸들 (☰) 클릭 → 원하는 위치로 이동
2. **"순서 저장"** 버튼 클릭

## 📝 기사 타입별 가이드

### 뉴스 (news)
- 일반 뉴스 기사
- 썸네일 URL 입력 가능

### YouTube (youtube)
- URL 형식: `https://www.youtube.com/watch?v=VIDEO_ID` 또는 `https://youtu.be/VIDEO_ID`
- 자동으로 iframe 임베드 생성

### Twitter (twitter)
- **Embed HTML** 생성: https://publish.twitter.com
- 트윗 URL 입력 → HTML 코드 복사 → "Embed HTML" 필드에 붙여넣기

### Instagram (instagram)
- **Embed HTML** 생성: Instagram 포스트 → "..." → "임베드"
- HTML 코드 복사 → "Embed HTML" 필드에 붙여넣기

## ✨ 하이라이트 기능

### 하이라이트 vs 일반

| 구분 | 하이라이트 (✅) | 일반 (❌) |
|------|----------------|-----------|
| 표시 형태 | 카드 형태 | 한 줄 텍스트 |
| 썸네일 | 표시 | 미표시 |
| 설명 | 전문 표시 | 미표시 |
| 타임라인 점 | 파란색 + 그림자 | 회색 |
| 아이콘 | 📌 표시 | 없음 |

### 하이라이트 사용 권장 사항
- ✅ 이슈의 핵심 전환점
- ✅ 주요 판결문, 성명서
- ✅ 영상 자료 (YouTube, Instagram)
- ✅ 전체의 30% 이하로 설정

## ❓ 자주 묻는 질문 (FAQ)

### Q. 후속 기사를 몇 개까지 추가할 수 있나요?
**A.** 제한은 없지만 **10-15개 이내** 권장합니다.

### Q. YouTube 영상이 표시되지 않아요.
**A.** 다음을 확인하세요:
- URL 형식이 올바른지 (`https://www.youtube.com/watch?v=...`)
- 영상이 비공개/삭제 상태가 아닌지
- 영상 ID가 11자리인지

### Q. Twitter/Instagram 임베드가 표시되지 않아요.
**A.** 다음을 확인하세요:
- Embed HTML을 올바르게 복사했는지
- 포스트가 삭제/비공개가 아닌지
- `<script>` 태그는 보안상 제거될 수 있음

### Q. 썸네일 이미지가 표시되지 않아요.
**A.** 다음을 확인하세요:
- URL이 유효하고 `https://`로 시작하는지
- 이미지 서버가 CORS를 허용하는지

### Q. 순서 변경이 저장되지 않아요.
**A.** 순서 변경 후 **"순서 저장" 버튼**을 반드시 클릭하세요.

## 💡 베스트 프랙티스

### 제목 작성
- 간결하고 명확하게 (50자 이내 권장)
- 핵심 내용을 앞에 배치
- 날짜/시간 정보 포함 (예: `[속보]`, `[11/4]`)

### 설명 작성
- 1-2문장으로 요약
- 중요한 사실만 언급
- 하이라이트 기사에만 설정

### 순서 관리
- 시간 순서대로 정렬 (최신이 아래)
- 하이라이트는 3-4개 간격으로 배치

### 썸네일 선택
- 고해상도 이미지 사용 (최소 800x600)
- 이슈 관련 대표 이미지 선택
- 저작권 확인

## 🔒 보안 안내

- **XSS 방어**: 시스템이 자동으로 악성 스크립트 제거 (`<script>` 태그 등)
- **안전한 태그만 허용**: DOMPurify 사용
- **개인정보 보호**: 개인정보 포함 기사 신중히 검토
- **저작권 확인**: 썸네일 이미지 저작권 확인 필수
